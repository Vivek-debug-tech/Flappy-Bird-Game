# Sudharma Flappy Bird Game

A fun Flappy Bird clone built with HTML5 Canvas, CSS3, and vanilla JavaScript.

## Features
- Smooth gameplay with physics-based bird movement
- Beautiful graphics and animations
- Sound effects for game start and game over
- Score tracking with local storage for best score
- Responsive design (works on desktop and mobile)
- Touch and keyboard controls

## How to Play
- Press **Space** or **Click/Tap** to make the bird jump
- Avoid hitting the pillars
- Try to beat your best score!

## Technologies Used
- HTML5 Canvas
- CSS3 (Glassmorphism effects)
- Vanilla JavaScript
- Google Fonts (Outfit)

## Local Development
Simply open `index.html` in a web browser to play!
